Element & Egenskaper

HTML-Element

Meta-taggar

  • <!DOCTYPE>: Denna deklaration specificerar vilken version av HTML som används i dokumentet.

  • <html>: Rotelementet i ett HTML-dokument som innehåller alla andra element.

  • <head>: En behållare för metadata om dokumentet, såsom titeln, länkar till stilmallar och metataggar.

  • <title>: Definierar titeln på HTML-dokumentet, som visas i webbläsarens titelrad eller flik.

  • <style>: Används för att bädda in CSS (Cascading Style Sheets) i ett HTML-dokument.

  • <link>: Specificerar externa resurser, som CSS-filer, att användas i dokumentet.

  • <body>: Huvudinnehållsområdet i HTML-dokumentet där text, bilder och annat innehåll visas.

Semantiska Element

  • <main>: Representerar huvudinnehållet i dokumentet, och det skall endast finnas ett <main> -element på varje HTML-sida.

  • <header>: Innehåller vanligtvis introducerande innehåll eller en webbplatslogotyp och navigation som rör hela webbplatsen.

  • <section>: Delar upp innehållet i avsnitt, ofta med en rubrik, och används för att gruppera relaterat innehåll.

  • <footer>: Innehåller vanligtvis information om företaget/skaparen, upphovsrätt eller kontaktuppgifter.

  • <arcticle>: Representerar en fristående del av innehåll som kan distribueras och återanvändas oberoende.

  • <aside>: Innehåller innehåll som är delvis relaterat till huvudinnehållet, ofta används för sidofält.

Text & Rubriker

  • <a>: Skapar hyperlänkar till andra webbsidor eller resurser.

  • <span>: En generisk behållare som används för att applicera stilar eller skript på en specifik del av texten.

  • <p>: Representerar ett stycke text.

  • <h1>...<h6>:Definierar olika nivåer av rubriker, med <h1> som den högsta och <h6> som den lägsta.

  • <ol>: Skapar en ordnad (numrerad) lista.

  • <ul>: Skapar en oordnad (punktlista) lista.

  • <li>: Definierar ett listobjekt inom en ordnad eller oordnad lista.

  • <strong>: Renderar text i fet stil men är reserverat för SEO-nyckelord.

  • <em>: Betonar text genom att rendera den i kursiv stil.

Layout

  • <div>: En allmän behållare som används för att gruppera och formge element, ofta för layoutändamål eller för att applicera CSS-stilar.

CSS-egenskaper

Typsnitt & stilar

  • font-family: Specificerar föredragna typsnittsfamiljer för text.

  • font-weight: Bestämmer tjockleken på tecken i ett typsnitt, som fetstil.

  • font-size: Anger storleken på texten inom ett element

  • text-decoration: Kontrollerar dekorationen av text, som underline, overline eller line-through

  • text-transform: Ändrar texten till versaler eller gemener

Färger

  • color: Definierar färgen på texten.

  • background-color: Sätter bakgrundsfärgen för ett element.

  • background-image: Sätter en linear-gradient eller bakgrundsbild.

  • background: Sätter en bakrundsfärg, linear-gradient eller bild.

  • linear-gradient(...): Skapar en övertoningsbakgrund för ett element.

Layout

  • position: Definierar ett elements positionssättning på webbsidan.

  • inset: Bestämmer avståndet från ett elements kant till dess föräldraelement.

  • top, right, bottom, left: Definierar avståndet från ett elements kant till dess föräldraelement, riktat mot respektive kant (överkant, högerkant, underkant, vänsterkant).

  • display: Anger hur ett element ska visas på webbsidan.

  • justify-content: Används med flexbox för att justera objekt längs huvudaxeln.

  • align-items: Används med flexbox/CSS Grid för att justera objekt längs tväraxeln.

  • flex-direction: Sätter avståndet mellan objekt i en flex-container.

  • text-align: Bestämmer justeringen av texten inom ett element.

  • gap: Avståndet mellan flexbox- eller CSS Grid-barn

Storlek & marginal

  • width: Specificerar bredden på ett element.

  • height: Bestämmer höjden på ett element.

  • max-width: Anger den maximala bredden som ett element kan ha.

  • min-height: Anger den minsta tillåtna höjden för ett element.

  • margin: Används för att justera avståndet utanför ett elements gränser.

  • padding: Bestämmer det inre avståndet mellan ett elements innehåll och dess gränser.

  • margin-inline: Sätter avståndet mellan objekt i en flex-container.Används för att justera det inre avståndet längs den inre axeln för ett blockelement.

Effekter

  • object-fit: Styr hur en bild eller video ska anpassas i sin box.

  • aspect-ratio: pecificerar en elements bredd-höjd-förhållande.

  • first-child, last-child, nth-child: Selektorer som väljer specifika element baserat på deras position i förhållande till deras föräldraelement.
